# Инструкция для работы с Markdown
## Выделение теста
Чтобы выделить текст курсивом необходимо обрамить его звездочками (*)
например, *вот так*
Чтобы выделить текст полужирным необходимо его обрамить текст двойными звездочками (**)
например, **вот так**
